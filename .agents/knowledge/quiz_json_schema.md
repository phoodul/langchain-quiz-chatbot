# 📊 Quiz JSON Schema Knowledge

이 지식은 퀴즈 생성 AI가 생성해야 하는 JSON 데이터의 구조를 정의합니다. 모든 퀴즈 생성 결과는 이 스키마를 엄격히 준수해야 합니다.

## 📐 JSON Structure

```json
{
  "question": "문제의 텍스트 내용",
  "options": [
    "1. 보기 1번",
    "2. 보기 2번",
    "3. 보기 3번",
    "4. 보기 4번"
  ],
  "answer": "정답의 숫자 번호 (예: 1, 2, 3, 4)",
  "explanation": "문제에 대한 상세 해설"
}
```

## ⚠️ 주의 사항
- **번호 포함**: `options` 리스트의 각 항목은 반드시 `1.` , `2.` 와 같이 번호로 시작해야 합니다.
- **정답 형식**: `answer`는 문자열이 아닌 숫자 형식을 선호하지만, 파서의 유연성을 위해 문자열 숫자로 응답할 수 있습니다.
- **독립성**: 각 문제는 문서의 맥락을 충분히 포함해야 하며, 이전 문제에 의존해서는 안 됩니다.
